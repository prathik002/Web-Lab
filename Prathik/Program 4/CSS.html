CSS


CSS, or Cascading Style Sheets, is a stylesheet language used to describe the presentation of a document written in a markup language, most commonly HTML. While HTML provides the structure and content of a webpage, CSS defines how that content should be displayed visually.


    There are three primary types of CSS:-
  1. Inline CSS
  2. Internal CSS
  3. External CSS

    CSS selectors:-
  1. CSS element Selector
  2. CSS id Selector
  3. CSS class Selector
  4. CSS Universal Selector
